# Alien-Invasion

A 'Space Invaders' like game where you play as a ship, and attempt to shoot all the aliens in the fleet before they get you.

## What can you do?

- Control the movements of the ship (left and right arrow keys)
- shoot the incoming fleet (spacebar)
- Keep a high score

## What's it built with?

Built in Python using Pygame: a Python library for creating games and multimedia applications: 

## How to run it on your computer

1. Download Python from python.org (when downloading make sure to check the box : "add python to path")
2. Verify installation by opening your terminal/command prompt and typing ``` python -- version ```
3. Install Pygame by typing ```pip install pygame``` or/and anyother libraries needed ```pip install [any other libraries]```
